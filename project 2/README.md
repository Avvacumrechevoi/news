# Интерактивная диаграмма Ганта для VK Видео

Профессиональная интерактивная диаграмма Ганта для управления проектом запуска новостного сценария в VK Видео с интеграцией Supabase для хранения данных в реальном времени.

## Возможности

### Реализованные функции

#### Базовый функционал
- **Трехуровневая иерархия**: Проект → Эпики → Задачи
- **9 эпиков и 36 задач** с полными данными о проекте VK Видео
- **Timeline с 12 месяцами** года для планирования
- **Цветовое кодирование** по типам работ (не по кварталам)
- **Сворачивание/разворачивание** эпиков одним кликом

#### Система статусов
- **3 статуса задач**: Ожидает, В работе, Выполнено
- **Визуальная индикация** статусов с цветными бейджами
- **Быстрое переключение** статусов одной кнопкой
- **Зачеркивание** выполненных задач
- **Затемнение** баров завершенных задач

#### Управление задачами
- **Добавление** новых задач в любой эпик
- **Редактирование** задач через модальное окно
- **Удаление** задач с подтверждением
- **Детальная информация**: название, описание, ответственный, даты, тип, статус

#### Аналитика и прогресс
- **Общий прогресс проекта** в процентах и счетчиках
- **Прогресс эпиков** с мини прогресс-барами
- **Автоматический пересчет** при изменении статусов
- **Ключевые цели** по кварталам Q1-Q4
- **Счетчики**: завершено, в работе, общий прогресс

#### Поиск и фильтрация
- **Полнотекстовый поиск** по названию, описанию, ответственному
- **Фильтрация по типу** задачи (Подготовка, Разработка, Запуск, Рост, Веха)
- **Фильтрация по статусу** (Ожидает, В работе, Выполнено)
- **Счетчик найденных** задач
- **Сброс фильтров** одной кнопкой

#### Экспорт данных
- **Экспорт в JSON** со всей структурой проекта
- **Экспорт в CSV** для работы в Excel/Google Sheets
- **Сохранение метаданных** о времени экспорта

#### Режимы просмотра
- **"Всё"** - показывает эпики и все задачи
- **"Только эпики"** - high-level обзор проекта
- **Массовые действия**: развернуть/свернуть всё

#### Интерактивность
- **Hover-эффекты** на задачах и эпиках
- **Анимации** при переключении статусов
- **Плавные переходы** и тени
- **Показ длительности** в днях при наведении
- **Визуальная обратная связь** для всех действий

#### База данных
- **Supabase PostgreSQL** для хранения данных
- **Real-time синхронизация** изменений
- **Автоматическая инициализация** данных при первом запуске
- **Row Level Security (RLS)** для безопасности
- **Публичный доступ** для демо-версии

## Технический стек

- **React 18** с функциональными компонентами и хуками
- **TypeScript** для типобезопасности
- **Tailwind CSS** для стилизации
- **Supabase** для базы данных и backend
- **Lucide React** для иконок
- **Vite** для сборки

## База данных

### Таблицы

**projects**
- id (uuid)
- name (text)
- description (text)
- created_at, updated_at (timestamptz)

**epics**
- id (uuid)
- project_id (uuid, FK)
- name, description (text)
- type (text): content, tech, integration, distribution, optimization, monetization
- start_month, duration (numeric)
- order_index (integer)
- created_at, updated_at (timestamptz)

**tasks**
- id (uuid)
- epic_id (uuid, FK)
- name, description, owner (text)
- start_month, duration (numeric)
- type (text): prep, dev, launch, growth, milestone
- status (text): pending, in-progress, done
- order_index (integer)
- created_at, updated_at (timestamptz)

## Использование

### Изменение статуса задачи
Кликните на кнопку с иконкой слева от задачи для циклического переключения:
- ○ Ожидает → ⊙ В работе → ✓ Выполнено → ○ Ожидает

### Добавление задачи
1. Кликните на кнопку "+" в строке эпика
2. Заполните форму в модальном окне
3. Нажмите "Сохранить"

### Редактирование задачи
1. Кликните на кнопку ✎ рядом с задачей
2. Измените нужные поля
3. Нажмите "Сохранить"

### Поиск задач
Введите текст в строку поиска для фильтрации по названию, описанию или ответственному

### Фильтрация
Выберите типы и статусы для отображения только нужных задач

### Экспорт данных
1. Наведите на кнопку "Экспорт"
2. Выберите формат (JSON или CSV)
3. Файл автоматически скачается

## Цветовая схема

**Типы задач:**
- Подготовка: голубой градиент
- Разработка: фиолетовый градиент
- Запуск: розовый градиент
- Рост: зеленый градиент
- Веха: оранжевый градиент (увеличенная высота)

**Типы эпиков:**
- content: голубой
- tech: фиолетовый
- integration: розовый
- distribution: оранжевый
- optimization: зеленый
- monetization: индиго
